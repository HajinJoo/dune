# DUNE - Modern Book Trailer Website

A professional, modern website about Frank Herbert's "Dune" - no spoilers. This is a sleek, premium-looking book trailer website with a sophisticated design aesthetic.

## Features

✨ **Modern Design**
- Clean, professional aesthetic inspired by luxury and mystery
- Desert-themed color palette (Gold, Black, Deep Tan)
- Responsive design that works on all devices
- Smooth animations and transitions
- Premium typography and spacing

🎯 **Sections**
1. **Navigation Bar** - Fixed, elegant navigation with smooth scroll
2. **Hero Section** - Striking introduction with the Dune title
3. **Story Timeline** - Six beautifully designed cards following the narrative arc
4. **World Information** - Learn about the universe without spoilers
5. **Call to Action** - Engagement buttons
6. **Footer** - Professional closing

## How to Use

1. **Open the Website**
   - Simply open `index.html` in your web browser
   - Works in Chrome, Firefox, Safari, Edge, etc.

2. **Navigation**
   - Click the navigation links to smoothly scroll to sections
   - Hover over buttons for interactive effects

3. **Responsive Design**
   - Website adapts beautifully to mobile, tablet, and desktop screens

## Customization Guide

### Colors
Edit the CSS variables in `styles.css` (lines 6-14):
```css
:root {
    --primary-color: #D4AF37;      /* Gold accents */
    --secondary-color: #1a1a1a;    /* Dark background */
    --accent-color: #8B7355;       /* Tan/brown accent */
    --text-light: #F5F5F5;         /* Light text */
    --text-dark: #1a1a1a;          /* Dark text */
}
```

### Content
- Edit the text in `index.html` directly
- Sections are clearly labeled with comments

### Fonts
Change the font in `styles.css` line 18:
```css
font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
```

### Add Images
To add background images or scene imagery:
1. Create an `images/` folder
2. Place your images there
3. Update CSS to reference: `background-image: url('images/filename.jpg');`
